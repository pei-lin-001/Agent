# Personal Agent Project Profile

This project is being adapted into pei-lin's all-purpose personal assistant.

## Current Direction

- Build from a reliable coding-agent foundation.
- Keep long-term memory automatic and invisible during normal use.
- Prefer local or self-hosted infrastructure for sensitive personal assistant capabilities.
- Store secrets in local secret stores, not tracked repository files.

## Architecture Preferences

- Add capabilities through extensions when possible.
- Keep stable profile context separate from dynamic long-term memory.
- Keep project-specific configuration reproducible with examples and scripts.
- Validate core behavior with focused regression tests and real end-to-end checks when practical.
